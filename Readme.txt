This repostory contains the MATLAB functions and the source code needed to implement the experiment presented in the following paper.

An artificial neural network ensemble approach to generate air pollution maps. 
S. Van Roode1, J.J. Ruiz-Aguilar, J. González-Enrique, I.J. Turias

Publications using this code, or derivative works, to produce figures or other content should cite this paper.

A sample database, output files and the necessary functions for the devepment are provided.
